Pipes:

As any other '.cfg' files: the pipes are pre-loaded. it is encouraged to trigger them. they are fun,and profitable for learning new languages/updating the desktop/and more.

--

  • pipe-translate.cfg: input:[$lang_id,$string] utilize: translate.sh

  • pipe-wallpaper.cfg input: A text file output: update the wallapaper
